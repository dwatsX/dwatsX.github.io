Affinity App

Affinity App

Affinity was our final capstone project at Conestoga College, and is a full-stack web app written in C# and the ASP.NET framework. Inspired by the social isolation and mental health issues surrounding the COVID-19 pandemic, the App is essentially a social media platform that matches users based on their interests. By matching people in this way, the goal is to create meaningful connections between people who have several things in common.


The App supports things like: creating a user profile, adding pictures, adding interests to your profile, matching with users who have interests in common, adding friends, private messaging, creating and joining groups, group messaging, and creating and joining group events.

Utilization of Bootstrap

In order to ensure that the site was useable on any device, the front-end of the site was designed using the Bootstrap framework, allowing it to stretch and collapse to fit on desktop, tablet, and mobile environments. The screenshot to the right shows the Profile page, in desktop and in mobile view.

Adding Friends

Once a user is matched with another user based on their common interests, the user can be added as a friend. Once the friend request is accepted, users are able to message each other with direct messages.


All data, including the status of user relationships, user profile data, messages, group data and event data, was all stored in an SQL database.

Direct Messaging

Building the messaging system from back to front was a challenging and rewarding experience. I tried to create an easy to use interface, similar to a standard SMS Message interface that you'd see used on a mobile phone.


Space Force

Space Force

Space Force is a game I created in my third semester at Conestoga College, in PROG2370 (Game Programming with Data Structures). The game was written entirely in C#, using the MonoGame framework.
Space Force is a classic arcade-style, top-down shooter, reminiscent of Space Invaders. It is a challenging game that gets progressively more difficult over time. In addition, players have the ability to upgrade their weapons, until they have powered up enough to use their ultimate ability!
Creating this game taught me a great deal about object-oriented programming, and it was a lot of fun too!
